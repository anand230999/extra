To run the code first you have to copy that ewo files to a folder of your machine.
Then run the below code on your Terminal:

  $ express <folder_directory>
  $ npm install socket.io

After that run the code from the folder directory by 

  $ nodejs chat.js
  
Now open your web browser and enter this address on your address bar 192.168.0.4:8000.
